Proxy
=====

### proxy_enable

Enable HTTP proxy.

Default: `false`

```php
public $proxy_enable = true;
```

### proxy_host

The HTTP proxy hostname or IP address.

```php
public $proxy_host = 'example.com';
```

### proxy_port

HTTP proxy port.

```php
public $proxy_port = 8080;
```

### proxy_user

HTTP proxy user.

```php
public $proxy_user = 'proxy_user';
```

### proxy_pass

HTTP proxy password.

```php
public $proxy_pass = 'proxy_password';
```
