title: Component Development Tutorial
---

This is a multi-step tutorial which will take you through the development of a Joomla component. 
It is aimed at people who have a working knowledge of PHP and JavaScript, 
but have no experience in developing components for Joomla.

If you are completely new to Joomla extension development
then please do the [Module Development Tutorial](../../modules/module-development-tutorial/index.md) first
as it introduces some key Joomla concepts which you'll need here, 
but which won't be explained in such detail in this tutorial.

All the code is available in the [Joomla manual-examples github repository](https://github.com/joomla/manual-examples/tree/main/component-tutorial),
but you'll benefit more from following the instructions to create the code yourself,
and use the github code only if you can't get the functionality to work.

Some tutorial steps suggest additional functionality which you can try to build yourself.
There aren't any answers supplied for these challenges,
but you would be able to get help via the [Joomla Community](../../../get-started/index.md#community).

If you don't already use an [IDE for PHP development](../../../get-started/ide/index.md) 
then you should consider installing one and becoming familiar with it,
as it will make your coding and debugging much easier in the long run.

We hope you find the tutorial useful, and enjoy going through it!
