# Language strings

File: `administrator/language/en-GB/com_content.ini`

Strings will be removed in 6.0: 
* `COM_CONTENT_RUN_TRANSITION`, use `JWORKFLOW_RUN_TRANSITION` instead 
* `COM_CONTENT_WORKFLOW_STAGE`, use `JWORKFLOW_STAGE` instead
